# COMP4920-Project
## instruction:
### frontend:
```sh
$ cd frontend
```
run the following command first if package.json was modified
```sh
$ npm install
```
otherwise run the following command straightaway:
```sh
$ npm run dev
```

### backend:
```sh
$ cd backend 
$ python manage.py runserver
```
